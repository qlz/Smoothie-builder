## GitHub Pages

This repository is set up to deploy to GitHub Pages with a GitHub Actions workflow.

After pushing these changes, the site will publish from the `main` branch workflow to:
[https://qlz.github.io/Smoothie-builder/](https://qlz.github.io/Smoothie-builder/)

If this is the first Pages deploy for the repository, GitHub may require one quick setup step:

1. Open `Settings > Pages`
2. Under `Build and deployment`, set `Source` to `GitHub Actions`

After that, every push to `main` will redeploy the site automatically.
